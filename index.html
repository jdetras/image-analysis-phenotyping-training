Who this course is for

Researchers, technicians, graduate students, and breeding-program staff who collect plant images — from flatbed scans and growth-chamber cameras to field drones — and want to turn them into quantitative, analysis-ready traits. The course assumes:

  • Basic Python literacy — you can read a script, run a function, and edit variables. A pre-course primer notebook is provided for refreshers.
  • A scientific background in plant biology, agronomy, or breeding — or a computational background working alongside plant scientists.
  • No prior computer-vision or deep-learning experience required. We build it up from first principles.

Cohort size is tuned for 16–24 participants with one instructor and one teaching assistant, so everyone gets hands-on support during practicals.

Learning outcomes

By the end of the week, participants will be able to:

01

Acquire & manage

Design an imaging protocol, control for illumination and scale, and organise datasets with reproducible metadata.

02

Pre-process

Apply denoising, colour-space transforms, and calibration so downstream measurements are robust and comparable.

03

Segment

Separate plant from background using thresholding, classical, and learned segmentation — and know when each fails.

04

Quantify shape

Extract morphometric traits (area, length, convex hull, skeleton, colour indices) and validate them against ground truth.

05

Apply classical ML

Engineer features and train interpretable classifiers/regressors for trait prediction and quality control.

06

Train deep models

Fine-tune CNNs, detectors (YOLO), and promptable segmenters (SAM) for counting, detection, and pixel labelling.

07

Reconstruct in 3D

Build point clouds and meshes via photogrammetry and measure volumetric/architectural traits.

08

Work reproducibly

Package an analysis in a versioned environment with notebooks, so a colleague can re-run it end to end.

Format & logistics

Daily rhythm

Mornings are concept + guided demo (~3 h). Afternoons are hands-on practicals on your own data or provided datasets (~3.5 h), with TA support.

Delivery

In-person computer lab or hybrid via video + shared notebooks. All compute can run locally or on free Google Colab GPUs.

Assessment

Daily practical checkpoints plus a Day-5 capstone applied to a real phenotyping question. Optional certificate on completion.

Getting started

  1. Read the Setup & pre-arrival checklist at least three days before the course.
  2. Install the conda environment (or use the Colab fallback) and run the verification notebook.
  3. Skim the datasets & references so you know what we'll be working with.
